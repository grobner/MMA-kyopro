## テンプレートの種類

### inv
逆元の計算のテンプレートです。繰り返し二乗法を用いて計算しています。
### choose1
解説資料の二項係数の計算(1)を用いて計算しています。
### choose2
解説資料の二項係数の計算(2)を用いて計算しています。
